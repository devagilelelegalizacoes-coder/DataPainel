Política de Privacidade

Última atualização: {{ hoje }}

1. Dados coletados

  • Dados de cadastro: nome, e-mail, senha (armazenada de forma criptografada), tipo de profissional, CNPJ ou nº de carteirinha de despachante e o documento comprobatório enviado.
  • Dados de uso: histórico de consultas realizadas, placas consultadas, créditos utilizados e resultados retornados.
  • Dados de pagamento: registros de compra de créditos (processados pelo Mercado Pago; não armazenamos dados de cartão).

2. Finalidade

Os dados são usados para: validar e aprovar o cadastro do usuário, viabilizar a execução das consultas solicitadas, manter histórico para consulta posterior pelo próprio usuário e cumprir obrigações legais.

3. Armazenamento e segurança

Os dados são armazenados em banco próprio, com senha protegida por hash e documentos comprimidos antes do armazenamento. O acesso a documentos de cadastro e anexos de consulta é restrito a operadores e administradores autorizados.

4. Compartilhamento

Os dados de identificação do veículo consultado são enviados aos provedores de dados veiculares (APIBrasil e bases parceiras) apenas para a execução da consulta solicitada. Não compartilhamos dados de cadastro do usuário com terceiros para fins comerciais.

5. Direitos do titular

O usuário pode, a qualquer momento, solicitar a atualização ou exclusão de seus dados de cadastro entrando em contato com a administração do sistema.

6. Contato

Dúvidas sobre esta política podem ser enviadas ao administrador da conta responsável pela contratação do serviço.
